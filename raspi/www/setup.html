🔒 Home Network Certificate Setup

Install the certificate to access your services securely

Step 1: Download Certificate

Click the button below to download the root certificate

📥 Download Certificate

Step 2: Install on Your Device

🪟 Windows
1
Double-click the downloaded certificate
The certificate file is named "caddy-root-ca.crt"
2
Click "Install Certificate"
Select "Local Machine" when prompted
3
Choose certificate store
Select "Place all certificates in the following store" and browse to "Trusted Root Certification Authorities"
4
Complete the wizard and restart your browser
Click "Next" → "Finish" → Restart Chrome/Edge/Firefox
🍎 macOS
1
Double-click the downloaded certificate
This will open Keychain Access
2
Click "Add" to install the certificate
The certificate will be added to your login keychain
3
Open Keychain Access
Applications → Utilities → Keychain Access
4
Find and trust the certificate
Search for "Caddy Local Authority" → Double-click → Expand "Trust" → Set "When using this certificate" to "Always Trust"
5
Enter your password when prompted
Close the window and restart your browser
🐧 Linux
1
Download the certificate via terminal
wget http://cert.home.arpa -O caddy-root-ca.crt
2
Install system-wide (Ubuntu/Debian)
sudo cp caddy-root-ca.crt /usr/local/share/ca-certificates/ sudo update-ca-certificates
3
Install for Firefox (if using Firefox)
certutil -A -n "Caddy Local Authority" -t "C,," -i caddy-root-ca.crt -d sql:$HOME/.mozilla/firefox/*.default-release
Note for Fedora/RHEL: Use sudo cp caddy-root-ca.crt /etc/pki/ca-trust/source/anchors/ && sudo update-ca-trust
